# 📌 프로젝트 기획서: 텐먼스 추세 트래커 (10-Month Trend Tracker)

## 1. 프로젝트 개요
* **프로젝트명:** 텐먼스 추세 트래커 (10-Month Trend Tracker)
* **기획 배경:** 일반 투자자들은 명확한 매수/매도 기준이 없어 상승장과 하락장에서 모두 손실을 봅니다. 하루 종일 호가창을 볼 수 없는 직장인들을 위해, 한 달에 딱 한 번 확인하여 기계적으로 대응할 수 있는 시스템이 필요합니다.
* **목적:** 성승현 작가의 '월봉 종가 매매법'을 자동화하여, 감정을 배제하고 기준선(10개월선) 돌파 여부만으로 안전하게 우량주 장기 투자를 할 수 있도록 돕습니다.

---

## 2. 타겟 유저 및 핵심 가치
* **타겟 유저:** 본업에 충실하며 짜투리 시간(월말)을 활용해 안정적인 수익을 내고 싶은 직장인 및 장기 투자자.
* **핵심 가치:** * **시간 절약:** 매일 장 마감 후 시장을 분석할 필요 없이, 매월 말일 1회 알림만 확인.
  * **감정 통제:** '저승사자 캔들(추세 이탈)' 발생 시 즉각적인 매도 경고를 통해 계좌 방어.
  * **명확성:** 보조지표를 배제하고 오직 가격(캔들), 추세(10개월선), 거래량에만 집중.

---

## 3. 핵심 스크리닝 로직 (알고리즘)

추세를 확실히 타는 종목을 걸러내기 위해, KOSPI 및 KOSDAQ 우량주(시가총액 상위 200위 이내)를 1차 필터링 풀(Pool)로 설정합니다.

| 구분 | 조건식 | 설명 |
| :--- | :--- | :--- |
| **기준 지표** | KOSPI/KOSDAQ, 시총 상위 200위 | 잡주를 배제하고 추세가 잘 만들어지는 종목만 선정 |
| **타임 프레임** | 월봉 (Monthly Chart), 종가 기준 | 일일 변동성(노이즈) 무시 |
| **매수 신호 (신규 진입)** | 전월 종가 < 전월 10개월선 **AND**<br>당월 종가 > 당월 10개월선 | 오랜 박스권이나 하락을 끝내고 **10개월선을 상향 돌파**하는 시점 |
| **보유 신호 (홀딩)** | 당월 종가 > 당월 10개월선 | 기준선을 깨지 않았으므로 계속 보유 |
| **매도 신호 (위험 경고)** | 전월 종가 > 전월 10개월선 **AND**<br>당월 종가 < 당월 10개월선 | 지지선이 무너진 상태. **저승사자 캔들 경고** (즉시 매도) |
| **보조 지표** | 당월 거래량 > 전월 거래량 (선택 사항) | 거래량이 실린 돌파인지 확인하여 신뢰도 상승 |

## 4. 시스템 구성도 및 기술 스택
* **데이터 수집 (Data Sourcing):** `FinanceDataReader` 또는 `OpenAPI(키움/한국투자증권)`를 활용하여 국내 주식 일봉/월봉 데이터 수집.
* **데이터 전처리 및 분석 (Processing):** `Python` 및 `Pandas` 라이브러리를 활용해 월봉 변환 및 10개월선(MA10) 연산.
* **알림 전송 (Notification):** `python-telegram-bot` API를 활용하여 텔레그램 메시지 발송.
* **자동화 스케줄링 (Automation):** `AWS EC2`, `Oracle Cloud(무료 티어)`, 또는 `PythonAnywhere`와 같은 클라우드 서버에 코드를 올리고 `Crontab`을 통해 **매월 말일 오후 3시 20분(장 마감 10분 전)**에 자동 실행.

## 6. 프로젝트 개발 로드맵

* **Phase 1 (데이터 수집 및 로직 검증):** 파이썬 환경 세팅, 시가총액 200위 리스트업, 월봉 데이터 변환 및 10개월선 연산 함수 개발.
* **Phase 2 (텔레그램 봇 연동):** Telegram BotFather를 통한 봇 생성, API 연동, 메세지 템플릿 작성 및 테스트 발송.
* **Phase 3 (서버 배포 및 자동화):** 클라우드 서버 환경 구축, 정기 실행 스케줄러(Cron) 등록, 예외 처리(공휴일 휴장, 상장폐지 종목 등) 로직 추가.
* **Phase 4 (고도화 - 선택):** 나스닥/S&P 500 등 미국 시장 대장주 스크리닝 기능 추가, 관심 종목만 따로 등록해서 트래킹하는 기능 추가.
